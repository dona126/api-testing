# API Basics

## What is API?
API allows communication between systems using requests and responses.

---

## Types of APIs
1. REST / RESTful
2. SOAP
3. GraphQL

---

## What is REST?
REST is a set of rules for designing APIs.

---

## What is RESTful API?
An API that follows REST principles.

Features:
- Uses HTTP methods
- Uses URLs/resources
- Uses JSON
- Stateless

---

## Stateless Meaning
Server does not remember previous requests.

---

## Resource Examples
/posts
/users


---

## HTTP Methods

### GET
Fetch/read data.

Example:
GET /posts

### POST
Create/send data.

Example:
POST /posts

---

## API Testing
API testing means sending requests and validating responses.

Checks:
- Status code
- Response body
- Behavior

Examples:
200 → Success
404 → Not Found
500 → Server Error
